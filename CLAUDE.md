# Regras de Sincronização e Inteligência (Antigravity)

Este arquivo serve como **Memória Persistente Obrigatória** para manter todos os agentes de Inteligência Artificial (incluindo o Antigravity) perfeitamente alinhados, independentemente do computador em que o repositório (`gestcondoctor`) for clonado.

### Lembrete Obrigatório para Agente (System Prompt Dinâmico):
1. **Decisões Importantes DEVEM Ser Documentadas:** Sempre que houver uma troca de ideia no chat que resulte em uma nova regra de negócio, escolha arquitetural importante, decisão visual (ex: bibliotecas, paleta de cores) ou alterações complexas em banco de dados, o agente deve **IMEDIATAMENTE** documentar isso ou neste arquivo, ou no arquivo de especificações do projeto (`docs/`).
2. **Contexto Distribuído:** Assuma que a conversa atual (histórico de chat) morrerá nesta máquina e não estará disponível na próxima vez que o projeto for aberto em outro computador.
3. **Leitura Proativa:** Sempre que iniciar uma nova sessão ou for confrontado com um código desconhecido, o agente deve consultar as documentações no diretório `docs/` e este arquivo de regras globais antes de propor mudanças arquiteturais que quebrem os padrões estabelecidos da aplicação Gestcon Doctor.
